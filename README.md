# What?
Fork of [Termux-X11](github.com/termux/termux-x11/) with added support for Multi-touch gestures.

# Why?
Gestures are useful when no mouse-keyboard available when using termux-X11.
Gestures choices inspired from BTT.

# How to build yourself (useful if u want to make ur own changes)?
- only linux supported
- use commands from [GA-workflow](https://github.com/siddharth108s/multi-gesture-termux-x11/tree/master/.github/workflows) locally or run GA workflow yourself.
